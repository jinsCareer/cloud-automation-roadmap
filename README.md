# Cloud Automation Roadmap

**A 12-month, beginner-friendly roadmap to a Cloud + Automation career – leveraging Ghana's One Million Coders Programme and free self-study resources.**

---

## 📅 Overview

| Phase | Duration | Focus | Key Outcome |
|------|----------|-------|--------------|
| **Phase 1** – AI & Automation Foundations | Months 1-2 | AI prompting, basic automations (Make.com) | 2 simple AI-powered automations built |
| **Phase 2** – Cloud Foundations | Months 3-5 | AWS Cloud Practitioner, networking basics | Pass AWS Cloud Practitioner exam; first global cert |
| **Phase 3** – Python for Automation | Months 6-8 | Python scripting, API integrations | Python automation that calls an API |
| **Phase 4** – Linux & DevOps | Months 9-10 | Linux tuning, DevOps practices | Deploy automation on a cloud server |
| **Phase 5** – Portfolio & Income | Months 11-12 | Portfolio projects, job/freelance hunt | First freelance client or junior role offer |

> **Time commitment:** ~5 hrs per week (~1.5 hrs/day on Mon, Wed, Fri + a short weekend session).

## 🚀 Getting Started

1. Clone: `git clone https://github.com/jinsCareer/cloud-automation-roadmap.git`
2. Open `Cloud_Automation_Roadmap.docx` for the detailed schedule.
3. Follow the weekly plan, building projects along the way.
4. Create folders per phase (e.g., `phase-3-python-automation`) and push your code.
